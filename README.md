## Start Project

Installing dependencies to run the project
```bash
    npm run prepare
```

Run manager , server and client
```bash
    npm run start:all
```

Run server 
```bash
    npm run start:server
```

Run server & manager
```bash
    npm run start:manager
```

Run server & client
```bash
    npm run start:client
```
